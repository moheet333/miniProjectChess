Chess Duels

First Player (Host)

  1. Open Chess Platform: Access the online chess platform.
  2. Enter Room Code: Create a unique game code
  3. Create Room: Look for the "Create Room" option.
  4. Wait for Second Player: Wait for the second player to join the room.

Second Player (Joiner)

  1. Open Chess Platform: Access the same platform.
  2. Enter Room Code: Input the code provided by the first player.
  3. Join Room: Click "Join Room" to join the room.

Starting the Game

  1. Once both players are in the same room, the first player is White, and the second player is Black.
  2. Start the game using the platform's interface and play following standard chess rules.
